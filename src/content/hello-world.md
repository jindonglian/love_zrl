## 欢迎

这是我的个人网站的第一篇文章。这个网站使用 **Vite + React + TypeScript** 构建，采用 **Tailwind CSS** 做样式，博客文章用 **Markdown** 编写。

## 为什么建这个网站

需要一个地方记录学习和思考。一条推文太短，一个 GitHub issue 太零散，一个博客刚好。

## 技术栈

- **构建工具**: Vite
- **框架**: React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **路由**: React Router v7
- **Markdown**: react-markdown + remark-gfm + gray-matter
- **部署**: GitHub Pages
